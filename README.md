## Project Name: 

**KeenKeeper — Keep Your Friendships Alive**

## Description:

This is a responsive website where people can manage their friendships. Users can make audio and video calls, and send text messages to their friends. They can also track their call and message history with their friends.

## Technologies Used:

- Next.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React Toastify
- JSON
- Recharts
- App router(Next.js)

## Features
- Smart Timeline Filtering
- Interactive Friendship Anlytics with Pie Chart Visualization
- Real Time Communication
